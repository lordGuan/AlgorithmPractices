# 连续正整数组成的字符串中寻找目标

## 问题
目标字符串：'1234567891011121314...' -- 正整数连续不断拼接的无现长字符串
输入：num:string 长度2-15 
输出：num在目标字符串中首次出现的位置

## 用例
```
 findPosition("456") == 3
 because "123456789101112131415".indexOf("456") = 3
             ^^^
```
